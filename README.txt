Weartherapp project

The idea of the project is to create a program for obtaining weather information from various weather sites.

The purpose of the project - the creation of a program that will collect information from the specified sites, will be easy to use and will be able to convert to the service.

Проект "Weatherapp"

Ідея проекту - створення програми для отримання інформації про погоду з різних погодніх сайтів.

Ціль проекту - створення програми, яка буде збирати інформацію з заданих сайтів, буде простою у користуванні та буде мати можливість перетворення у сервіс.
